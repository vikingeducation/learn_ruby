Diving Into Markup and Coding

Welcome to the final mini-course in our prep program!

This is where the rubber meets the road. Over the last 3 mini-courses, you've learned:

  • How the web works
  • How to use your command line
  • How to use the Git version control system
  • How to think user-first for good UX
  • How to design a page which directs the user through its flow
  • How to build site mockups in Balsamiq
  • The basic tenets of engineering problem solving
  • How the Agile Development process helps developers manage projects
  • How to design good modular solutions to complex problems and write them using pseudocode.

As we've said all along, having the proper context and background for what you're building is very important in order to do it right and to create something that people actually want.

What You'll Learn

All the lessons in the list above will begin to circle back together in this mini-course, where we're going to teach you how to actually build websites with HTML markup and to write server-side code in Ruby. It's no doubt what you've been waiting for!

By the end of this mini-course, you'll have picked up two important skill sets, one on the front end and one on the server side.

First, you'll learn the tools you need to create a website using HTML and to style it with CSS. This alone will enable you to build your own static websites using many existing tools across the web.

But you're presumably here because you want to build real full-featured web applications, not just static sites. That's why the last section will teach you how to solve problems with the Ruby programming language. Ruby is used by the Ruby on Rails framework which powers the servers of hundreds of high growth startups and companies around the world. Rails also forms the core of our intensive program curriculum.

Even if you're not here as prep for the intensive program, what you learn over the coming lessons will help you kick start your web development abilities and will give you tools that you can use to climb to the next step in your technical education. This is where the pace picks up and things get really fun!

We hope you enjoy it!
